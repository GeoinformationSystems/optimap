## Codebook

| Field                   | Description                                             |
|-------------------------|---------------------------------------------------------|
| `id`                    | Primary key of the publication record                   |
| `title`                 | Title of the article                                    |
| `abstract`              | Abstract or summary                                     |
| `doi`                   | Digital Object Identifier (if available)                |
| `url`                   | URL to the article or preprint                          |
| `publicationDate`       | Date of publication (ISO format)                        |
| `creationDate`          | Record creation timestamp (ISO datetime)                |
| `lastUpdate`            | Last update timestamp (ISO datetime)                    |
| `geometry`              | Spatial geometry in GeoJSON/WKT                         |
| `timeperiod_startdate`  | List of coverage start dates (ISO format)               |
| `timeperiod_enddate`    | List of coverage end dates (ISO format)                 |
| `provenance`            | Source or method by which the record was imported/added |
